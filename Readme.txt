## About
This is an event management APP with which user can login to the app, Create, Read, Update and Delete Events.


1. Authentication Using JWT,
2. Session Management. And auto logout during inactive sessions
3. Backend API built using Express
4. Frontend with React
5. React Browser Router for Routing, 
6. Implemented Nested Routings, Loader and actions


## How to start Backend

cd backend 
npm start

## How to start Frontend
cd frontend
npm start
